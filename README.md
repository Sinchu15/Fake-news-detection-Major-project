# Fake News Detection System

A Machine Learning based web application that detects whether a news article is Fake or Real using Natural Language Processing (NLP).

## Features
- Fake vs Real news prediction
- Text preprocessing using NLP
- TF-IDF feature extraction
- Naive Bayes Machine Learning model
- Flask web application
- User login page
- Graph visualization of dataset

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS
- Matplotlib
- Google Colab
- VS Code

7. Web Application Deployment

## How to Run
1. Install required libraries: pip install flask pandas scikit-learn matplotlib
2. Run the project : python app.py
3. Open in browser:http://127.0.0.1:5000
