# Fake News Detection System

Fake News Detection System is a Machine Learning based web application that detects whether a news article is Fake or Real using NLP and Flask.

## Features
- Fake vs Real news prediction
- Login page
- Graph visualization
- Web-based interface

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- Matplotlib
- HTML/CSS

## Dataset
The dataset is large and cannot uploaded to Github

## How to Run

1. Install required libraries

```text
pip install flask pandas scikit-learn matplotlib
```

2. Run the project

```text
python app.py
```

3. Open browser

```text
http://127.0.0.1:5000
```

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Text Cleaning
4. Feature Extraction using TF-IDF
5. Model Training
6. Prediction
7. Web Application Deployment

## Output
- Detects Fake or Real news
- Displays prediction result
- Shows dataset graph
